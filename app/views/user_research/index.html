Prototype for user research

This page contains the prototype for features that require user research in order to better understand the users need and subsequently the approach to design we should be taking.

View mortgage deed flow

  • Viewing mortgage deed - happy path
    • mortgage deed - nobody confirmed
    • mortgage deed - other borrower confirmed

Unhappy paths

  • Borrower reference / DOB does not match a deed
  • Invalid date of birth
  • Deed processing has been stopped
  • Invalid authentication code

Errors

  • Server error - 404 - page not found
  • Server error - 5xx - unexpected error
  • Session has timed out or ended
  • Your deed has not been confirmed
